No releases yet — building toward 0.10.0

Freally Player is in the planning stage; the application itself is not built yet. When the first release ships, signed installers for Windows, macOS, and Linux will appear here, and this page will list each version's changes. Freally Player is completely free — every feature included: the full player, the media library, casting, and conversion. No ads, no telemetry, no account — ever.

What's coming first (0.10.0 — Foundation)

  • The Tauri v2 shell (Rust core + a modern React/TypeScript UI in the Havoc dark theme), with the Windows build as a GUI app that opens no console window.
  • The key architecture: decoded video rendered into a native, hardware-accelerated surface composited under the web UI (via the libmpv render API) — proven on Windows, macOS, and Linux before any features are built.
  • The cross-platform CI matrix and a tag-triggered release workflow that produces draft installers per OS.
  • This Havoc-branded site, with the roadmap to 1.0.0.

Roadmap to 1.0.0

  1. 0.10.0
    Foundation & the native video surface

    Tauri + Rust shell and hardware-accelerated video on a native surface under the modern web UI.

    Up next
  2. 0.20.0 – 0.50.0
    Core playback, subtitles, audio, playlists

    Glassy controls + resume; full subtitles (ASS + online fetch); audiophile audio (EQ/exclusive/passthrough/tracks); playlists, video filters, mini-player/PiP.

    Coming soon
  3. 0.60.0 – 0.70.0
    Streaming, URLs & discs

    Network streams; YouTube/site URLs via yt-dlp; webcams, internet radio, podcasts; DVD/Blu-ray/ISO menus (no decryption keys bundled).

    Coming soon
  4. 0.85.0
    Media library — first public release

    Folder scan, opt-in scraping (posters/metadata), continue-watching, music library, and signed installers. The modern VLC, with a library.

    Milestone
  5. 0.95.0
    Casting & conversion

    Cast to / be a Chromecast/DLNA renderer — whole queues included; conversion/clip/record, instant remux, and GIF export.

    Coming soon
  6. 1.0.0
    Polish, themes, i18n & launch

    Skins/themes, accessibility (keyboard-first, screen-reader, high-contrast), and the full 18-language UI with RTL Arabic.

    Coming soon
  7. Beyond 1.0
    Power-user & format reach

    Exploring: shader/upscaler profiles, parametric EQ / room correction, secondary subtitles, collections/box-sets, and a 10-foot HTPC mode — all local, all opt-in.
